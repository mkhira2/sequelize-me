# Sequelize-Me

Just a simple 'hello world' app using sequelize.

## Installation

```
git clone https://github.com/mkhira2/sequelize-me.git
cd sequelize-me
npm install

createdb [database-name]

edit config/config.json with your specs (username, [database-name])

node app.js
```
